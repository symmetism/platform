# Platform / apps

General-purpose Symmetism apps live here (visualizers, generic tooling,
the symmetism.com platform itself). Apps that are *about* Reflexivity
(stabilizer validators, framework-tied visualizations) live in
`Reflexivity/apps/` instead.

When unclear, default here and move later via `git mv` (history preserved).
See `_command/02_SOT_RULES.md` R6 for the full placement rule.
